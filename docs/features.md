# 策略风险收益评估及对比

## 特性描述
根据策略的每日收益率信息，评估其在不同时间范围的风险、收益，做对比分析

## 功能点
- 收益和回撤曲线: 根据用户选择的时间范围, 绘制不同策略的收益曲线和回撤曲线
- 时间范围: 包括全部、最近 5 年、最近 3 年、最近 1 年、YTD(今年至今) 以及用户自定义范围
- 基准: 用户可选择不同的指数作为基准
- 风险收益评估: 根据用户选择的时间范围，评估不同策略的风险收益指标，包括年化收益率, 年化波动率， 超额年化收益率，超额年化波动率，夏普率，最大回撤, 索提诺比率，卡玛比率，信息比例（超额年化收益率/超额年化波动率), 阿尔法系数，贝塔系数等。用表格的形式展示，并高亮每个指标最优的是哪个策略。
- 最近收益情况: 以表格形式展示，不同策略最新的净值，WTD( 本周累计收益率), MTD(本月累计收益率), YTD( 本年累计收益率), 本年最大回撤，当前回撤。

## 要求

- 每日收益率数据当前支持 CSV 格式（参考@sample_data/pnl.csv），以后考虑其他格式
- UI 界面采用多页面的形式，不要改变之前设计的页面
